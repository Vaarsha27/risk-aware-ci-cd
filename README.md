# Risk-Aware CI/CD Optimization 🚀

## 📌 Overview
This project predicts risks in CI/CD pipelines using Machine Learning.

## 🧠 Features
- Predict pipeline failure risk
- Uses numerical + text data
- ML model with TF-IDF + scaling

## 🛠 Tech Stack

- NumPy, Scikit-learn
- Joblib

- predict.py
